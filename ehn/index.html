EHN ／ 3つの機能貼る ・ 共感する ・ 照らす
FUNCTION 01

貼る

見積もりを匿名で投稿。写真でもPDFでもいい。あなたの一枚が、誰かの物差しになる。

業者名・施主名・電話・住所・会社印は、掲載前にこちらで必ず伏せます。
FUNCTION 02

共感する

同じ悩みにコメント。「うちもそうやった」がここで繋がる。一人で抱えなくていい。

コメントは確認のうえ掲載。荒らし・個人名の晒しは載せません。
FUNCTION 03

照らす

その見積もりが適正かを、フラッグで照らす。判断材料が、その場で一目でわかる。
