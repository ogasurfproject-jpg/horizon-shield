EHN（見積りハッカーニュース）は、日本中の見積りが集まる、完全無料の掲示板。
あなたの一枚を貼る。誰かの一枚が見える。それだけのことが、今までどこにも無かった。

社是 ・ なぜ集めるのか
一枚ずつ集まるほど、みんなの物差しが、太くなる。
これは告発の場やない。物差しを、みんなで養う場所や。
＋あなたが貼る → コーパスに一枚増える
＋誰かが貼る → あなたが比べられる一枚が増える
＋コメントが付く → 「うちもや」が繋がる
＝日本中の物差しが、一本太くなる
同じ80㎡・外壁塗装でも、これだけ違う（例）
A 社168万
B 社240万
C 社398万
EHN ／ 3つの機能 貼る ・ 共感する ・ 照らす
FUNCTION 01

貼る

見積りを匿名で投稿。写真でもPDFでもいい。あなたの一枚が、誰かの物差しになる。

業者名・施主名・電話・住所・会社印は、掲載前にこちらで必ず伏せます。
FUNCTION 02

共感する

同じ悩みにコメント。「うちもそうやった」がここで繋がる。一人で抱えなくていい。

コメントは確認のうえ掲載。荒らし・個人名の晒しは載せません。
FUNCTION 03

照らす

その見積りが適正かを、フラッグで照らす。赤か、青か。判断材料が、その場で一目でわかる。

業者を裁くためやない。あなたが自分の物差しを持つためのフラッグです。
