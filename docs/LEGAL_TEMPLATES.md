## 1. 建設業法の基本リスクと対応

### 1.1 リスクの本質
建設業法第3条：軽微な建設工事以外の請負には建設業許可が必要。施主がHSの提示価格を「HSと契約して工事を依頼したい」と誤認すると、HSが**無許可営業**と見なされる可能性がある。

### 1.2 HORIZON SHIELDの立ち位置（明確化必須）
- **HSは施工を請け負わない**
- **HSは施主と施工業者の間に立たない（仲介業者ではない）**
- **HSが提供するのは「相場に基づく参考価格の情報提供」のみ**
- **施主はHSの情報を元に、自分で施工業者と直接契約する**

### 1.3 回避の3原則

1. **「見積もり」という言葉を避ける** → 「参考価格提示」「相場情報」と表記
2. **金額は必ずレンジで示す**（確定額ではない）
3. **免責事項を全ての出力物に記載**（UI・PDF・メール・LINEトーク）

## 3. 表記ルール（UI/PDF/配布物共通）

### 3.1 使ってよい言葉 ✅

- 参考価格
- 相場レンジ
- 市場相場に基づく目安
- 業者交渉の参考資料
- HS基準（HSが算出した業者規模別の妥当な利益率レンジ）
- 適正価格の参考情報

### 3.2 使ってはいけない言葉 🚫

- 見積書（正式な建設業法上のもの） → ×
  - 代わりに: 「参考見積書（相場分析レポート）」
- 契約金額 → ×
  - 代わりに: 「相場レンジ」「参考金額」
- 施工保証 → ×（HSは施工しない）
- 請け負います・施工します → 絶対NG
- 必ず安くなる・絶対〇〇万円安くできる → 誇大表示でNG（消費者契約法違反リスク）

### 3.3 グレー表現の注意

- 「業者の過剰請求」→ 〇（市場データとの乖離という客観事実）
- 「悪徳業者」→ △（特定業者を誹謗と見なされない範囲で）
- 「ぼったくり」→ △（比喩表現にとどめる・法的トラブル時のみ事例引用）

## 5. 消費者保護の倫理ガイドライン

### 5.1 TOshi哲学と法務の両立

**「業者を敵視しない」** → 客観的な相場データのみを提示。特定業者の誹謗中傷は記載しない。

**「施主を武装化する」** → 交渉材料としての相場情報は積極提供。ただし「必ず値下げできる」等の断定は避ける。

**「業者規模別の妥当利益率を透明化」** → 「大手だから悪い」ではなく、「大手の価格は区分4の妥当利益率レンジ」と表現。

### 5.2 脆弱な消費者（高齢者・認知症等）への配慮

- PDF内で自ら「家族・ケアマネ等への相談推奨」を明記
- 訪問販売業者からの急かし契約は消費者ホットライン188へ誘導
- バリアフリー工事（介護保険連動）は特に高齢者保護の明示

### 5.3 悪用防止

- HSのPDFを「これに合わせろ」と業者に一方的に突きつけるのは不適切
- PDFは「相談のきっかけ」「情報格差の是正」のために使用することを明記
- 業者が正当な理由で相場より高い見積もりを出すケースもある（現場条件等）

## 7. Red Flag 表示の倫理

### 7.1 OK: 客観的な業界警告として表示

```
⚠️ Red Flag 検出
このカテゴリでは以下の業者行為が国民生活センターに多数報告されています:
・「今すぐ契約しないと危険」等の即決迫り
・火災保険で無料と謳う訪問販売
...
```

### 7.2 NG: 特定業者への断定的な攻撃

```
❌「〇〇社の見積もりはぼったくりです」
❌「この業者は悪徳です」
❌「この金額は詐欺です」
```

→ 必ず一般論として表記。特定業者名は記載しない。
